Project 2: Markowitz Optimization

Solve for optimal weights:

def optimize_portfolio(returns):
    cov = np.cov(returns.T)
    inv_cov = np.linalg.inv(cov)
    ones = np.ones(len(cov))
    return inv_cov @ ones / (ones.T @ inv_cov @ ones)
